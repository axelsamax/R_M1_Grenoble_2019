Quels jeux ont eu le temps de jeu_moyen le plus lev des deux dernires semaines (top 20)?

video_games %>%

  top_n(20, average_playtime) %>%

  mutate(game = fct_reorder(game, average_playtime)) %>%
  
  ggplot(aes(x = game, y = average_playtime / 60)) +

  geom_point(color = "red" ,alpha = .5, size = 3) +

  coord_flip() +

  xlab(NULL) +

  ylab("Temps de jeu moyen (heure)") +

  labs(title = str_wrap("Temps de jeu moyen des deux dernires semaines", 25)) +

  labs(subtitle = "les 20 meilleurs jeux jous en moyenne sont affichs")

Ce code affiche les 20 jeux les plus joues cest deux dernires semaines. Le jeu le plus jou est Clicker Heroes plus de 80 heures en ses deux dernires semaines.

            ecart_type = sd(price, na.rm = T),
